# PyFBB

**PyFBB** is a pure-Python library implementing the classic **F6FBB** packet radio BBS forwarding protocol.

### Features
- Full ASCII and binary forwarding modes
- Authentic **LZHUF** compression (original FBB algorithm)
- Reverse forwarding support (`FR` command)
- Clean, well-tested API
- Comprehensive unit and integration tests
